# Calculator v2.0.7

シンプルで使いやすいGUI電卓アプリと、その専用インストーラーです。
軽量で高速に動作し、誰でも直感的に操作できます。

---

## 🌟 特徴 (Features)

### 🧮 電卓機能

* 四則演算（+ - × ÷）に対応
* GUIで直感的に操作可能
* 小数・負数の計算に対応
* 軽量で高速起動
* 完全オフラインで使用可能

### ⚙️ インストーラー機能

* **単体実行可能（Setup.exe）**

  * 1ファイルだけでインストール可能
* **Go:Embed テクノロジー**

  * アプリ本体を内蔵
* **UX設計**

  * 95msの進行演出で「インストール感」を強化
* **レトロモダンUI**

  * 青背景＋白文字のクラシックデザイン
  * 視認性の高いプログレスバー

## 🛠 インストール方法

1. `Setup.exe` を実行
2. 利用規約を確認
3. 同意してインストール
4. 指定した場所に `AkihabaraCalculator` フォルダが作成されます
